Git гэж юу вэ?

-Git гэдэг нь код бичих үед хийсэн өөрчлөлтүүдийг хадгалж, буцааж харах боломжтой “түүх хадгалагч” шиг систем юм. Мөн олон хүн нэг төсөл дээр зэрэг ажиллахад өөрчлөлтүүдийг нэгтгэх, зөрчил шийдэхэд тусалдаг.

"Энэхүү репо нь Git \& GitHub-ийн анхан шатны гэрийн даалгавар юм."
